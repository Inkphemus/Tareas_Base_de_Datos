Con PYTHON instalado en el sistema junto a su gestor de paquetes PIP ejecute el siguiente comando.

pip install -r requirements.txt

Lo anterior instalara todas las dependencias adicionales que se podrian nesesitar.

Luego basta ejeuctar el siguiente comando para iniciar el cliente.

python CLI.py

## -- ---------------------------------------------

El cliente se iniciara y solicitara las credenciales de algun usuario registrado para verificar su acceso,
solo si este es confirmado exitosamente se procedera a entregar 5 opciones en bucle infinito al usuario,
las cuales pueden finalizar con la opcion 5.
