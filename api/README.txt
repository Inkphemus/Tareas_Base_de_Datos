La API se inicia con el siguente comando en la terminal.

bun run API.ts

## -- -----------------------------------------

Basta que la "API.ts" se encuentre en el proyecto de PRISMA y el "schema.prisma" se configure como el adjunto en este repository.

Ademas si las dependencias de ELYSIA estan disponibles no habra problemas, de lo contrario se pueden espeficicar.
[visitar linea 240 de "API.ts" ]


Una vez iniciada la API esta, estara corriendo, para entregar sus servicios http, en la IP "localhost" y el PORT "3000".
Por su puesto, en cualquier momento, podra detener el programa, precionando las teclas "Ctrl" + "c" en la terminal.
